# DSM Racing Video Media Page

This package contains a standalone `index.html` page and web-friendly MP4 versions of the uploaded videos.

## Deploy
Upload the whole folder to GitHub and deploy with Vercel as a static site.

## QR Code
The QR code section automatically points to the live page URL after deployment.

## Social Links
Open `index.html` and replace the `#` links for YouTube, Instagram, TikTok, Facebook, merch, and race fund/donation.
